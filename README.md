# Zomato SQL Analysis

## Project Overview
This project analyzes Zomato restaurant data using SQL queries. The analysis was performed using MySQL Workbench.

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## Dataset
- Zomato Dataset

## SQL Queries
1. Top 5 restaurants by rating
2. Table booking availability
3. Count of restaurants by category
4. Average rating analysis
5. Restaurant count
6. Online order analysis
7. Dining category analysis
8. Other SQL insights
